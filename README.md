# NextHire 🚀  
Smart Job Application Tracker + AI Resume Analyzer  

---

## 📌 Overview
NextHire is a web-based platform that helps job seekers organize, track, and improve their job application process.

Instead of managing applications manually, users can store all their applications in one place, monitor their progress through different hiring stages, and gain insights into their job search performance.

The platform also includes an AI-powered Resume Analyzer that evaluates how well a resume matches a job description and provides actionable suggestions to improve it.

---

## 🎯 Problem Statement
Job seekers often:
- lose track of applications
- forget to follow up
- don’t understand why they get rejected
- struggle to tailor resumes for each job

NextHire solves these problems by combining tracking, analytics, and AI feedback into one system.

---

## Core Features

### Job Application Tracking
- Add, edit, delete applications
- Store company, role, salary, notes
- Track application status

### Pipeline Board
Visual representation of application stages:
- Applied
- Screening
- Interview
- Offer
- Rejected

---

### AI Resume Analyzer
- Upload resume (PDF)
- Paste job description
- Get:
  - Match score
  - Missing keywords
  - Improvement suggestions

---

### Analytics Dashboard
- Weekly application activity
- Offer rate
- Pipeline funnel
- Salary insights

---

### Reminders
- Set follow-up dates
- Track pending actions

---

### Search & Filter
- Filter by:
  - status
  - company
  - role

---

## User Flow
1. User registers or logs in
2. Adds job applications
3. Tracks status through pipeline
4. Uses AI Analyzer to improve resume
5. Monitors performance via analytics
6. Sets reminders for follow-ups
